## 📌 About the Project

This is a C++ console application that demonstrates Object-Oriented Programming (OOP) concepts using inheritance.

The system allows the user to register either an employee or a patient and displays their information.

---

## 🚀 Features

- 🧱 Base class: `Persona`
- 👤 Derived classes:
  - `Empleado`
  - `Paciente`
- 🔁 Dynamic user input
- 🖥️ Console output
- 🧩 Inheritance and encapsulation

---

## 🛠️ Technologies

- C++
- Standard Library (`iostream`, `string`)

---

## 💻 How it works

1. User chooses:
   - Employee (E)
   - Patient (P)
2. Inputs personal data
3. Inputs additional data:
   - Category (Employee)
   - Insurance (Patient)
4. Program displays the information
